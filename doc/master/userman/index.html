Proof General

This file documents version 4.6-git of Proof General, a generic Emacs interface for proof assistants.

Proof General 4.6-git has been tested with GNU Emacs 25.2 on Linux. It is supplied ready to use for the proof assistants Coq, EasyCrypt, qrhl-tool, and PhoX.
